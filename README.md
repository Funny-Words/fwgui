# fwgui

fwgui (FunnyWords GUI) is a simple GUI for interacting with FunnyWords API.

## Installation

See Releases page

### Compiling

Clone repository, install gtk-devel and gtk libraries.

Then do

```cr
crystal build src/fw.cr --release
```
